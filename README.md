## Snitch Clone Website

A clone of the Snitch e-commerce website built using HTML and CSS.

### Tech Stack
- HTML
- CSS

### Features
- Navbar with logo and search bar
- Auto-scrolling image banner
- Featured categories grid
- Product listings

### Known Limitations
- ⚠️ Not responsive (desktop only)
